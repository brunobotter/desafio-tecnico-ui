# DesafioTecnicoUI

## Criação de uma spa para consumir a api criada.

## Development server

Rodar a aplicação `ng serve`. Navegue para `http://localhost:4200/`. 

## Tecnologia utilizada

- Angular Material
- Formulario Reativos
- API JAVA
